monocle

Statusline visual

Elegí presets o armá una statusline custom por segmentos, sin escribir scripts bash a mano.

Hooks y settings

Gestioná hooks, permissions, output style, modelo, variables de entorno y servidores MCP desde una UI consistente.

Seguro y actualizable

Hace backup antes de escribir, detecta updates en GitHub y permite actualizar con monocle update.

Instalación

Un solo comando. Sin duplicar installers, sin inventar flujos raros.

curl -fsSL https://raw.githubusercontent.com/CogniDevAI/monocle/main/install.sh | sh

El script detecta OS y arquitectura, descarga la última release y deja el binario en ~/.local/bin.
